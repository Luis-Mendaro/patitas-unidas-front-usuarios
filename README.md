# Patitas Unidas — Frontend Usuarios

Aplicación web para conectar refugios de animales con personas interesadas en adoptar mascotas.

**Demo:** https://patitas-unidas-front-usuarios.vercel.app

---

## ¿Qué hace la app?

- Registro e inicio de sesión de usuarios
- Listado de mascotas con filtros
- Vista de detalle por mascota
- Formulario de adopción paso a paso
- Listado de refugios y perfil de cada uno
- Sección de mascotas guardadas (favoritos)
- Perfil de usuario

---

## Tecnologías

- React + Vite
- Redux
- JavaScript
- CSS

## Correr localmente

```bash
git clone https://github.com/Luis-Mendaro/patitas-unidas-front-usuarios.git
cd patitas-unidas-front-usuarios
npm install
cp .env.example .env   # configurar VITE_API_URL con la URL del backend
npm run dev
```
